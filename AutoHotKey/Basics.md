# 핫키 & 핫스트링

핫키는 사용자가 키보드를 눌러 어떤 행위를 촉발시키는 키 또는 키의 조합입니다.
핫스트링이란 무엇인가? 핫스트링은 주로 타자하는 동안 약자를 확대하는 데 사용

### 핫키

```
^j::
   Send, 나의 첫 번째 스크립트
Return
```

