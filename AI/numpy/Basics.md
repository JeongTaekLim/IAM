### 배열 slice

```
# np.array로 표현된 학생들의 성적표
scores_np = np.array([
    [80, 92, 70],
    [91, 75, 90],
    [86, 96, 42],
    [77, 92, 52],
    [75, 85, 85],
    [96, 90, 95]
])
print(scores_np[0:, :])
print(scores_np[0:, :1])
```

```
[92 75 96 92 85 90]
[[80]
 [91]
 [86]
 [77]
 [75]
 [96]]
```
