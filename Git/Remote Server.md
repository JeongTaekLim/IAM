# Remote Server

* fetch
```
$ git fetch
```
* checkout
```
$ git fetch
$ git checkout test
```

* check branch
```
// remote
$ git branch -r
// local
$ git branch
```

* show difference
```
$ git diff [fileName]
```

* add
```
// add all files
git add .
```

* reset
```
// reset all added files
git reset
```

* push
```
git push [repository name, ex. origin] [branch name, ex. feature/abc]
```
